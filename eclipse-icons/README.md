# Eclipse Icons

Minimal icon theme with Eclipse customizations built on Material Icon Theme.

## Features

- 2000+ icons from Material Icon Theme
- 5 Eclipse-specific customizations:
  - App.jsx/tsx with React atom
  - main/index entry files
  - \*.component.jsx/tsx files
  - components/ folder
