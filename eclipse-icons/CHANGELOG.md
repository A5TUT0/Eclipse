# Change Log

All notable changes to Eclipse Icons will be documented in this file.

## [0.0.3] - 2025-11-06

### Changed

- **BREAKING**: Now uses Catppuccin Icons as base
- Removed custom icons for common files (now inherited from Catppuccin)
- Kept only Eclipse-specific customizations (App, Main, Components)
- Added Catppuccin Icons as extension dependency
- Updated README to explain the inheritance model

### Benefits

- Access to 500+ professionally designed icons from Catppuccin
- Automatic updates when Catppuccin improves their icons
- Smaller package size (only custom Eclipse icons)
- Best of both worlds: Catppuccin quality + Eclipse customizations

## [0.0.2] - 2025-11-06

### Added

- Special icons for App.jsx, main.jsx, index.js
- Custom component folder icons
- React component specific icons (\*.component.jsx)
- More granular file type support

## [0.0.1] - 2025-11-06

### Added

- Initial release of Eclipse Icons
- File icons for JavaScript, TypeScript, Python, Java, HTML, CSS, JSON, Markdown, React, Vue
- Configuration file icons for Git, npm
- Special folder icons for src, node_modules, .git, .vscode
- Open and closed folder states
- Root folder indicators
- Vibrant color palette matching Eclipse Dark themes

### Features

- SVG-based icons for crisp display
- Language-specific coloring
- Smart folder recognition
- Lightweight package size
