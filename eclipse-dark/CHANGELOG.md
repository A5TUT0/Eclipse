# Change Log

All notable changes to the "Eclipse Dark" theme will be documented in this file.

## [0.0.6] - 2025-11-06

### Added

- Marketplace badges to README (version, downloads, rating, installs)
- FAQ section with common questions and troubleshooting
- Comprehensive .vscodeignore for smaller package size
- More keywords for better marketplace discoverability

### Fixed

- IMPORTANT tag color corrected to teal (#4ec9b0) instead of red
- Better documentation formatting

### Improved

- README structure and clarity
- Package optimization with updated .vscodeignore

## [0.0.5] - 2025-11-06

### Added

- **Extension Pack**: Now automatically installs Better Comments extension
- Better integration with Better Comments for enhanced comment tags
- Updated documentation with clear setup instructions

### Changed

- Extension category updated to include "Extension Packs"
- Description updated to mention Better Comments integration

## [0.0.4] - 2025-11-06

### Added

- **Special Comment Tags**: TODO, FIXME, HACK, NOTE, IMPORTANT, BUG, XXX now have unique colors and bold italic style
  - TODO: Bright blue (#82aaff / #7aa2f7)
  - FIXME: Orange (#ff9d6a / #ff9e64)
  - HACK: Purple (#d4a5ff / #bb9af7)
  - NOTE/IMPORTANT: Teal (#5dd7c2 / #4fd6be)
  - BUG/XXX: Red (#ff5555 / #f7768e)
- String escape characters now have bold styling and distinct colors
- Template expression highlighting in template strings
- Enhanced regex patterns with bold styling
- Color values (hex/rgb) now have bold purple styling

### Improved

- Comments are now more visible with better colors and italic style
- Comment documentation has better contrast (#9aa5ce)
- String colors are more vibrant (#a6e22e in Flat)
- Regular expressions are more readable with better contrast
- Overall theme personality with unique code annotations

## [0.0.2] - 2025-11-06

### Added

- **New Theme**: Eclipse Dark Flat - A completely flat design with uniform #23283d background
- Flat theme features vibrant cyan/blue accents (#82aaff)
- All panels and bars use the same base color for minimal distraction
- Enhanced flat theme with modern, clean aesthetics

### Improved

- Eclipse Dark Pro: More vibrant bracket pair colors
- Better color consistency across both themes
- Improved readability with enhanced foreground colors

## [0.0.1] - 2025-11-06

### Added

- Initial release of Eclipse Dark Pro theme
- Deep dark background for reduced eye strain
- Vibrant orange parameter highlighting (#ffa657)
- Progressive property coloring for chained properties
- Excellent contrast between code elements
- Modern color palette optimized for readability
- Support for semantic tokens
- Beautiful bracket pair colorization
- Enhanced terminal colors
